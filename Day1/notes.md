#Day 1 - Python Introduction

What is Python?
Python is an easy and simple programming language. It is used in software development, data science, machine learning.

How to display?
print() function is used to display output.
Example: 
print("Hello")
print(5)
It can print:
->text
->numbers
->multiple lines

How to get an input?
input() function is used to get input from an user.
Example:
a=input()
b=input("Enter your name")

File Extension:
Python file should saved within the extension .py
Example:
data.py
input.py

Today, I learned how to write and run Python program using the print() function.
