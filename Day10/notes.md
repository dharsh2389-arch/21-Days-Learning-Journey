Dictionary in Python

What is a Dictionary?
A dictionary is a collection of key-value pairs.
It is:
- Ordered
- Mutable (can be changed)
- Does not allow duplicate keys
Example:
student = {
    "name": "Dharshini",
    "age": 18,
    "department": "ECE"
}
