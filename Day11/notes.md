 What is a Function?
A reusable block of code that performs a task.

 Syntax:
 def function_name(parameters):
     statements
     return value
     
Types of Functions
Without parameters
With parameters
With return value
Default parameters

What is an Array?
An Array is a data structure used to store multiple values of the same data type in a single variables.

Difference between Array and List:
Array: Does not allow mixed number of types.
List: Allow mixed number of types.

Use Array when:
You need only numbers
Memory efficiency matters
Large numeric data processing

Use List when:
You need flexibility
Mixed data types
General programming tasks
