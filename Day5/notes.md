#Conditional Statements

What are Conditional Statements?
Conditional statements are used to make decisions in a program.
Python uses:
- if
- if else
- elif

1. if Statement:
Syntax:
if condition:
    statement
Example:
if 5 > 3:
    print("True")

2. if-else Statement:
Syntax:
if condition:
    statement
else:
    statement

3. elif Statement:
Used when there are multiple conditions.
Syntax:
if condition1:
    statement
elif condition2:
    statement
else:
    statement

4. Nested if:
An if inside another if.
